## 使用方式

1. **GitHub 瀏覽** — 直接在 GitHub 上讀
2. **NotebookLM 匯入** — 加入 Google NotebookLM 作為知識源，用自然語言查詢
3. **本機查詢** — Mickey 的 vault 系統直接引用
