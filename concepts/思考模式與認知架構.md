# 人的思考模式 × 心學 × 康德 × AI 認知架構

## 核心洞見

**心學關注「如何成為」，康德關注「如何認識」，AI 提供「如何實作」。**
三者整合，形成可成長人格系統的理論基礎。

---

## 一、思考模式演進

| 學派 | 核心觀點 | 思考路徑 |
|------|---------|---------|
| 亞里斯多德 | 人是理性動物 | 感知→抽象→概念→推理→結論 |
| 笛卡兒（理性） | 我思故我在 | 懷疑→清晰觀念→演繹→真理 |
| 洛克/休謨（經驗） | 心靈白板 | 感官→聯想→習慣→判斷 |
| 康德（整合） | 先天範疇+經驗 | 感官材料+先天結構→綜合判斷 |
| 現象學 | 意向性、在世存有 | 存在先於本質，自由創造自我 |

## 二、王陽明心學三核心

1. **心即理** — 理不在外物，在於心之本體
2. **致良知** — 良知是內在道德直覺
3. **知行合一** — 若未行則未真知

## 三、心學 vs 康德

| 範疇 | 王陽明 | 康德 |
|------|--------|------|
| 核心問題 | 人如何成為 | 人如何認識 |
| 道德基礎 | 良知直覺 | 義務倫理 |
| 思考完成於 | 行動 | 判斷 |

## 四、AI 認知架構對照

- **傳統 AI**：輸入→編碼→語義理解→推理→輸出
- **康德式 AI**：先天模型架構 + 訓練資料 → 知識生成
- **心學式 AI**：Value Core（良知）→ 道德評估 → 行動生成

## 五、人格 + 良知 + Memory + 決策 模型

```
本體核心（Value Heart）
    ↓
良知引擎（Moral Engine）
    ↓
感知 → 記憶 → 推理
    ↓
決策生成 → 行動
    ↓
回饋更新（Memory + Value 校正）
```
